# Run Length Encoding (RLE)
Lossless
- replaces consecutive repeated values with a single value and a count
- Effective for data with long sequences of identical values
Pros
- lossless
- simple and fast
- effective for long sequences of same
Cons
- might be larger in cases where there arent really repeated values
used in PNG,PDF, etc.
