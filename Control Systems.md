# Input, Process, Output
### Input
- Data or signals recieved from sensors or user interfaces
- initiates system actions
- eg temperature
### Process
- control logic or algorithm that processes input data
- determines output
- typically executed by a controller, eg microcontroller or CPU
- uses predefined rules or computations
### Output
- Actions or signals generated by the system
- To achieve desired effect
- sent to actuators or devices

### Feedback Mechanisms

##### Open Loop
No feedback - system operates without monitoring the output
eg. washing machine - you set the settings and it washes for set time
it doesn't check whether the clothes are washed

Input doesn't change depending on output
##### Closed Loop
- uses feedback to monitor output and adjust inputs for accuracy
- A aircon adjusts airflow depending on room temperature based on continuous temperature feeback
- Feeback ensures system accuracy and adaptability with closed loop systems being more precise but complex


# Actuators and Transducers
### Actuator
Device that perform physical actions based on controller output - motors, valves, heaters

### Transducers
Convert one energy to another form
eg. electical sensors

eg. converts heat to electrical signal for temp sensing
