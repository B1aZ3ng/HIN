# GPU
A graphics processing unit (GPU) is a specialised electronic circuit designed to accelerate the rendering of images, videos and animations by performing rapid mathematical calculations.

GPUs contain thousands of smaller cores making it better for parallel processing
many cores performing simple calculations
Designed for simultaneous execution of multiple tasks, ideal for data-intensive computations
### Features
Specialised Design - optimised for massive data processing
Parallel Processing - Handles large datasets concurrently, such as rendering 3d graphics or matrix multiplication for AI
VRAM - video ram
High throughput
